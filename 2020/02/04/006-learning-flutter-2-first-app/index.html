Flutter学习笔记2 - 第一个App
Created 2020-02-04| Updated 2020-02-04
Post View:|comments:

Flutter环境安装好之后，迫不及待的想要尝试一下了吧，下面介绍三种创建Flutter项目的方式。

方式一：命令行

  1. 创建项目

    flutter create hello_world

  2. 启动一个模拟器，以iOS为例

    open -a Simulator

  3. 运行项目

    cd hello_world
fullter run

方式二：Android Studio

  1. 下载Android Studio

  2. 安装Flutter和Dart插件

  3. Start a new Flutter project

  4. 启动一个模拟器

  5. 点击运行按钮启动程序

方式三：VS Code

  1. 下载VS Code

  2. 安装Flutter和Dart插件

  3. 快捷键Command+Shift+P，选择Flutter: New Project

  4. 启动一个模拟器

  5. 点击运行按钮启动程序
